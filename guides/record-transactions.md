# 거래 기록 (매수 / 매도 / 배당)

보유 종목을 탭하여 개별 종목 상세 페이지로 진입합니다. 페이지 중앙에 **매수**, **매도**, **배당** 세 가지 버튼이 있어 각 거래를 개별적으로 기록할 수 있습니다.

> **팁:** 매수 원가는 **가중평균법**으로 계산됩니다. 매수할 때마다 시스템이 자동으로 평균 원가(수수료 포함)를 재계산합니다.

# 보유 종목 추가

주식 계좌 상세 페이지에서 우측 상단의 **＋** 버튼을 탭한 후, 종목 코드·매수 수량·단가를 입력하여 포지션을 만듭니다.

1. 주식 계좌 상세 페이지에서 우측 상단의 \*\*＋\*\*를 탭합니다.<br>
2. 종목 코드 (예: 005930)와 종목명 (예: 삼성전자)을 입력합니다.<br>
3. 매수 수량과 단가를 입력합니다.<br>
4. **저장**을 탭하면 해당 종목이 보유 목록에 나타납니다.
